## Commit convention
```javascript
Type: Subject
ex) Feat: 회원가입 API 추가

Description

Footer 
ex) Resolves: #1, #2
```
### Types

- Feat: 기능 추가, 삭제, 변경
- Fix: 버그 수정
- Refactor: 코드 리팩토링
- Style: 코드 형식, 정렬 등의 변경. 동작에 영향 x
- Test: 테스트 코드 추가, 삭제 변경
- Docs: 문서 추가 삭제 변경. 코드 수정 x
- Etc: 위에 해당하지 않는 모든 변경
